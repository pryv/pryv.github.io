Value Types Directory

activity

An activity is defined by its time, duration, and classification. Thus, it has no specific value per-say.
nametypedescription
activity : pryvnullPryv activity event. No value here, as the value is actually defined by the event's time and duration fields as well as its classification (folder, tags).

notes

This is a generic type for all kinds of text notes a user may want to save. It includes subtypes such as standard text, html text, and more complex subtypes like Facebook events, etc.
nametypedescriptioncommentspropertiesmaxLength
notes : facebookobjectA Facebook event. start_time & end_time must be set into event's data & duration Facebook Event can be copied RAW as a value.. map -> fbEvent.name -> pryvEvent.comment fbEvent.start_time -> pryvEvent.time fbEvent.end_time - fbEvent.start_time -> pryvEvent.duration TODO: what to do with videos & pictures?see row below
