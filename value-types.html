Value Types

activity

An activity is defined by its time, duration, and classification. Thus, it has no specific value per-say.
nametypedescription
activity:pryvnullPryv activity event. No value here, as the value is actually defined by the event's time and duration fields as well as its classification (folder, tags).
properties: -

count

A simple number of items (e.g. eggs, apples).
nametype
count:genericnumber

notes

This is a generic type for all kinds of text notes a user may want to save. It includes subtypes such as standard text, html text, and more complex subtypes like Facebook posts, etc.
nametypedescriptioncommentsmaxLength
notes:facebookobjectA Facebook post. Facebook post can be copied RAW as a value. map: fbPost.message -> pryvEvent.description fbPost.created_time -> pryvEvent.time TODO: what to do with videos & pictures?

        "required": "true"

    "from": {
        "type": "object",
        "properties": {
            "name": {

            },
            "id": {

                "required": "true"

        }
    },
    "to": {

        "properties": {

                "type": "array",

                    "properties": {
                        "name": "string",
                        "id": "string"
                    }
                }
            }
        }
    },
    "message": {
        "type": "string"
    },
    "picture": {
        "type": "string"
    },
    "link": {
        "type": "string"
    },
    "caption": {
        "type": "string"

    "description": {
        "type": "string"

    "source": {

        "type": "string"
